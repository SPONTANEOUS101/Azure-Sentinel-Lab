# Azure Sentinel Learning Lab

This repository documents my end-to-end Azure Sentinel project, including:

- Workspace setup
- Log ingestion (AuditLogs + SignInLogs)
- KQL queries
- Detection rules
- Automation rules
- Playbooks
- Incident investigation workflow

## Repository Structure

- /detections — Analytics rules with explanations
- /automation — Playbooks + automation rules
- /queries — KQL queries used in detections
- /architecture — Setup documentation

## Goal

To build a beginner-friendly SOC pipeline using Microsoft Sentinel and learn:

- KQL fundamentals
- Detection engineering
- Incident automation
- Identity security monitoring
